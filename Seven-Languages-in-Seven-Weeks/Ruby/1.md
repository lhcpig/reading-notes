
### 4.打印字符串"This is sentence number 1."，其中的数字1会一直变化到10。
```ruby
10.times do |i|
    puts "This is sentence number #{i+1}."
end
```

`#{i+1}`可以用`(i+1).to_s`替换
也可以这样：

```ruby
1.upto(10) do |i|
    puts "This is sentence number #{i}."
end
```

### 6.加分题：如果你感觉意犹未尽，还可以写一个选随机数的程序。该程序让玩家猜随机数是多少，并告诉玩家是猜大了还是猜小了。
```ruby
puts "Guess a number:"
a = gets()
b = rand(10)
while true
   if a.to_i > b
      puts "Too large\n"
   elsif a.to_i < b
      puts "Too small\n"
   else
      puts "all right\n"
      exit
   end
   puts "Guess again:\n"
   a = gets()
end
```
